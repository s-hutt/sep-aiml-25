shapiq_student.copula¬∂

Implementation of the Gaussian copula imputer.

This module implements a gaussian conditional imputer using the Gaussian copula transformation. It enables imputing missing features in a feature coalition by modeling dependencies using a Gaussian copula approach, and generating conditional samples for SHAP or other feature attribution methods. (For conventional gaussian approach see gaussian.py)

Gaussian copula-based conditional imputer for missing feature imputation.

This class implements conditional sampling based on a Gaussian copula transformation. It is used for imputing missing features in a way that respects the dependence structure of the observed data. This is particularly useful in model explanation tasks such as computing SHAP values.

  • x (np.ndarray | None) ‚Äì Data points to explain. If None, will be inferred.

  • normalize (bool) ‚Äì Whether to normalize using the empty prediction.

Initialize the GaussianImputer.

This class performs conditional imputation based on a multivariate Gaussian distribution fitted to the background data. It is primarily used for imputing missing features (as defined by coalitions) during model explanation tasks.

Parameters:

  • model (callable) ‚Äì The predictive model to explain. Must support .predict() with 2D input.

  • data (np.ndarray) ‚Äì Background dataset used to estimate the feature distribution (n_samples, n_features).

  • x (np.ndarray or None, optional) ‚Äì The instance(s) to explain. If not provided during initialization, can be set later.

  • sample_size (int, default=10) ‚Äì Number of Monte Carlo samples to generate per coalition.

  • conditional_budget (int, default=128) ‚Äì Reserved for future use or budget-aware strategies (not directly used here).

  • conditional_threshold (float, default=0.05) ‚Äì Reserved for future use, such as feature pruning based on contribution (not used here).

  • normalize (bool, default=True) ‚Äì Whether to normalize output values by the empty prediction value.

  • categorical_features (list of int or None, optional) ‚Äì Indices of categorical features. Gaussian imputation does not support categoricals.

  • method ({'gaussConditional'}, default='gaussConditional') ‚Äì The imputation method name. Only ‚ÄògaussConditional‚Äô is supported in this class.

  • random_state (int or None, optional) ‚Äì Random seed for reproducibility.
